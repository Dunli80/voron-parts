# Undermount z chain using Klicky
Using the [Klicky Probe](https://github.com/jlas1/Klicky-Probe) mount 
as the upper support for the z chain making use of the smaller chain from
the x or y gantry.

- X or Y gantry chain
- M5 roll-in and bolt
- M3 roll-in and bolt
